# react-native-mechanic-map

React Native wrapper for mechanic map

## Installation

```sh
npm install react-native-mechanic-map
```

## Usage

```js
import MechanicMap from "react-native-mechanic-map";

// ...

const result = await MechanicMap.multiply(3, 7);
```
